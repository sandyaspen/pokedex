# Pokédex

A Pokémon search and browsing app built with React, Vite, and Tailwind CSS.

## Features
- Search for any Pokémon by name
- Browse and filter the original 151
- View base stats and types

## Tech Stack
- React
- Vite
- Tailwind CSS v4
- React Router
- PokéAPI
